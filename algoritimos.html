Método Insertion Sort

Clique no botão "Gerar"
para uma matriz aleatória ser gerada.

Clique no botão "Ordenar"
para matriz ficar ordenada ordenada.

Clique no botão "Ordenar Inverso."
para a matriz ficar ordenada ao contrário
